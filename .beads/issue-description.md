## Goal
Create a stunning, professional landing page that explains the option value of AI-enabled parallel exploration.

## Design Direction
- Inspiration: Financial Times, The Economist
- Fonts: Serif for headlines, clean sans for body
- Colors: Off-white/cream background, dark text, minimal accent
- Layout: Generous white space, clear hierarchy, premium feel
- Feel: Modern, crisp, trustworthy, financial-world professional

## Page Sections
1. Header - Minimal, just title
2. Hero - Big headline, compelling subhead
3. The Problem - Traditional dev = forced bets
4. The Theory - Real options explained simply
5. The AI Shift - Exploration cost collapsed, moonshots unlocked
6. Calculator - Interactive tool with sliders and chart
7. Case Studies - 2-3 placeholder sections
8. References - Academic citations
9. Footer - Minimal

## Technical Requirements
- Single HTML file + CSS + JS (no build step)
- Responsive (desktop-first, mobile-friendly)
- Calculator with sliders, real-time updates
- Chart.js via CDN for visualizations
- Good og:meta for LinkedIn sharing

## Deliverables
- index.html
- style.css
- calculator.js
- Placeholder content (real content comes later)
